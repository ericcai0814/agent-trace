# docs/agents/

Per-repo dev config consumed by [`mattpocock/skills`](https://github.com/mattpocock/skills) — issue tracker, triage labels, and domain doc layout.

**Contents are gitignored.** The files in this directory are author-specific dev environment configuration, not part of the public `agent-trace` surface. Only this README is committed so the directory structure stays visible.

If you want to use mattpocock engineering skills in your own clone, invoke their `setup-matt-pocock-skills` skill — it will populate this directory for your environment.
